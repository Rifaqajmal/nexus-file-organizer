# Nexus File Organizer

Nexus File Organizer is a desktop application developed in **Python** that automatically organizes files inside a selected folder based on different criteria such as file type, date, alphabetical order, or file size.

The application provides a graphical user interface (GUI) that helps users manage large collections of files efficiently.

---

## Features

* Organize files by file type
* Organize files by date
* Organize files alphabetically
* Organize files by file size
* Automatic folder creation
* Simple graphical user interface

---

## Technologies Used

* Python
* CustomTkinter
* Pandas
* Matplotlib
* Pillow (PIL)

---

## How to Run

Run the Python file:

```
python NexusFile_Pro.py
```

Or run the provided executable:

```
NexusFile Pro Advanced.exe
```
